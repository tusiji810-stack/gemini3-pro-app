# Gemini 3 Pro APP - 快速开始指南

## 🚀 云编译方式（推荐）

### 步骤1：安装 Git
下载并安装 Git：
https://git-scm.com/download/win

### 步骤2：创建 GitHub 仓库
1. 打开 https://github.com/new
2. 仓库名称填写：`gemini3-pro-app`
3. 选择 `Public`（公开）
4. 点击 `Create repository`

### 步骤3：上传代码
双击运行 `push_to_github.bat`，按提示操作。

或者手动执行：
```bash
cd gemini3_pro_app
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/你的用户名/gemini3-pro-app.git
git push -u origin main
```

### 步骤4：等待自动构建
1. 打开你的 GitHub 仓库页面
2. 点击 `Actions` 标签
3. 等待构建完成（约5分钟）

### 步骤5：下载APK
构建完成后：
1. 点击 `Actions` → 最新的构建记录
2. 页面底部 `Artifacts` 部分
3. 下载 `release-apk` 文件
4. 解压得到 `app-release.apk`

### 步骤6：安装到小米14
1. 将APK发送到手机
2. 在手机上点击安装
3. 允许未知来源应用安装
4. 开始使用！

---

## 📱 应用配置

首次使用需要配置API：

1. 打开APP
2. 点击右上角 ⚙️ 设置图标
3. 展开 `API配置` 卡片
4. 输入你的 API Key
5. 点击 `保存配置`

---

## ⚠️ 注意事项

1. **GitHub Actions 免费额度**：每月2000分钟，足够本项目使用
2. **构建时间**：首次约5-8分钟，后续约3分钟
3. **APK大小**：约20-30MB
4. **网络要求**：APP需要连接互联网使用API

## 🐛 常见问题

### 构建失败
- 检查代码是否有语法错误
- 查看GitHub Actions日志定位问题

### APK安装失败
- 确保允许安装未知来源应用
- 检查手机存储空间

### API连接失败
- 检查网络连接
- 确认API Key有效
- 检查API地址是否正确
