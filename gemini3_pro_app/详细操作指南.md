## 步骤3：获取GitHub Token（用于身份验证）

由于GitHub安全要求，推送代码需要使用Token而不是密码。

### 3.1 打开Token设置
访问：https://github.com/settings/tokens

### 3.2 创建新Token
1. 点击右上角 `Generate new token (classic)`
2. 可能需要输入密码验证

### 3.3 填写Token信息
```
Note: Gemini3Pro
Expiration: No expiration (永不过期)
☑ repo  ← 必须勾选这个（允许访问仓库）
```

### 3.4 生成并复制Token
1. 点击页面底部的 `Generate token`
2. **立即复制生成的Token**（绿色背景的一串字符）
   - 格式类似：`ghp_xxxxxxxxxxxxxxxxxxxx`
   - ⚠️ 注意：这个Token只显示一次！

## 步骤5：等待自动构建

推送成功后：

1. 打开您的仓库页面
   ```
   https://github.com/你的用户名/gemini3-pro-app
   ```

2. 点击顶部的 `Actions` 标签

3. 您会看到正在进行的构建任务
   - 黄色圆圈：正在构建（约5-8分钟）
   - 绿色勾：构建成功
   - 红色叉：构建失败

### 构建过程说明
```
Build Android APK
├── Setup Java            ✓ (30秒)
├── Setup Flutter         ✓ (1分钟)
├── Get dependencies      ✓ (1分钟)
├── Analyze project       ✓ (30秒)
├── Build APK             ✓ (2-3分钟)
└── Upload artifacts      ✓ (30秒)
```

## 步骤6：下载APK

构建成功后，有两种方式下载：

### 方式1：Artifacts下载（推荐）
1. 点击成功的构建记录
2. 页面底部找到 `Artifacts` 部分
3. 点击 `release-apk` 下载
4. 解压下载的zip文件
5. 得到 `app-release.apk`

### 方式2：Releases页面
1. 点击仓库的 `Releases`（右侧边栏）
2. 下载最新版本的APK

## 步骤7：安装到小米14

1. 将 `app-release.apk` 发送到手机（微信、QQ、数据线）
2. 在手机上点击APK文件
3. 可能提示"未知来源应用"：
   - 点击 `设置`
   - 开启 `允许来自此来源的应用`
4. 点击 `安装`
5. 完成！

## 步骤8：配置APP

1. 打开 `Gemini 3 Pro` APP
2. 点击右上角 ⚙️ 设置图标
3. 展开 `API配置` 卡片
4. 输入您的 API Key
5. 点击 `保存配置`
6. 开始使用！

## 🆘 遇到问题？

### 推送失败：Authentication failed
- 确保使用了Token而不是密码
- Token需要有 `repo` 权限

### 推送失败：Repository not found
- 检查仓库名称是否拼写正确
- 确保仓库已经创建

### 构建失败
- 点击Actions查看详细日志
- 可能是代码问题，需要修复

### 下载后找不到APK
- Artifacts下载的是zip文件，需要解压
- 解压后得到 app-release.apk
