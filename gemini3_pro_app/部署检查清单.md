# Gemini 3 Pro APP - 部署检查清单

## ✅ 部署前检查

### 1. GitHub 账号
- [ ] 已有 GitHub 账号
- [ ] 知道 GitHub 用户名

### 2. 创建仓库
访问：https://github.com/new
- [ ] Repository name: `gemini3-pro-app`
- [ ] Description: `Gemini 3 Pro 图像生成器安卓APP` (可选)
- [ ] 选择 `Public` (公开)
- [ ] 勾选 `Add a README file` (可选)
- [ ] 点击 `Create repository`

### 3. 运行部署脚本
双击运行：`一键部署.bat`
- [ ] 按提示输入 GitHub 用户名
- [ ] 输入邮箱（首次使用Git）
- [ ] 选择分支名称（推荐 main）

### 4. 身份验证
如果提示输入密码：
- [ ] 输入 GitHub Personal Access Token
- [ ] **不是** GitHub 登录密码

获取 Token：https://github.com/settings/tokens
- 点击 `Generate new token (classic)`
- Note: `Gemini3Pro`
- Expiration: `No expiration`
- 勾选 `repo` 权限
- 点击 `Generate token`
- 复制生成的 token（注意：只显示一次！）

## 🚀 部署后操作

### 5. 等待构建
- [ ] 打开 https://github.com/你的用户名/gemini3-pro-app
- [ ] 点击 `Actions` 标签
- [ ] 等待显示绿色 ✓ (约5-8分钟)

### 6. 下载APK
方式一：Artifacts
- [ ] 点击最新的构建记录
- [ ] 页面底部找到 `Artifacts`
- [ ] 点击 `release-apk` 下载
- [ ] 解压 zip 文件得到 `app-release.apk`

方式二：Releases
- [ ] 点击仓库页面的 `Releases`
- [ ] 下载最新版本的APK

### 7. 安装到手机
- [ ] 将APK发送到小米14
- [ ] 在文件管理器中点击APK
- [ ] 允许"安装未知应用"
- [ ] 完成安装

### 8. 配置APP
- [ ] 打开 Gemini 3 Pro APP
- [ ] 点击右上角设置 ⚙️
- [ ] 展开 `API配置`
- [ ] 输入 API Key
- [ ] 点击 `保存配置`
- [ ] 返回主页面开始使用

## 🐛 常见问题

### 推送失败
**错误信息：** `remote: Repository not found`
- 原因：仓库不存在
- 解决：先在GitHub创建仓库

**错误信息：** `Authentication failed`
- 原因：密码错误或需要Token
- 解决：使用 Personal Access Token 代替密码

### 构建失败
- 点击 `Actions` → 失败的构建 → 查看日志
- 常见原因：代码语法错误
- 解决：修复代码后重新推送

### APK安装失败
- 确保允许"未知来源"安装
- 检查手机存储空间
- 确保下载完整（APK约20-30MB）
